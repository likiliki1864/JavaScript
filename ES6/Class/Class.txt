Class:

*Introduces a blueprint for creating objects in JavaScript. Enables inheritance and constructor functions.
    Syntax:

        class ClassName {
            constructor() { }
            method() { }
        }
    Example:
        class Person {
        constructor(name) {
            this.name = name;
        }
        greet() {
            return `Hello, ${this.name}`;
        }
        }

        const person = new Person('LIKITH');
        console.log(person.greet()); // "Hello, LIKITH"
